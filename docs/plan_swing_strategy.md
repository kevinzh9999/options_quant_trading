# 项目计划：日线波段策略

> 目标：在日内策略的"盲区"（低振幅单边趋势期）赚取多日持仓利润。与日内策略互补。
>
> 创建：2026-04-04

---

## 核心理念

**日内策略亏钱的日子 = 波段策略赚钱的日子。**

215天数据验证：
- 日内策略在两段单边趋势（6/23~8/25 +23%, 12/17~1/12 +14.7%）只捕获了7%的涨幅
- 理想多日持仓可捕获：+1,400pt + +1,068pt = +2,468pt
- 这两段如果能捕获50%，就是 +1,234pt 增量

| 市场状态 | 日内策略 | 波段策略 | 识别方法 |
|----------|---------|---------|---------|
| 高振幅震荡 | +49/天 | 不参与 | 日内振幅>1.5% |
| 低振幅趋势 | -27/天 | 赚钱 | 日内振幅<1.0% + 方向明确 |
| 中等振幅 | +2/天 | 按信号 | 1.0-1.5% |

## Phase 1：趋势识别研究（1-2天）

### 1.1 日线技术指标回算

用 `index_daily`（2015~2026）计算：
- 20日均线（MA20）
- 布林带（20日，2σ）
- ADX(14)
- Hurst(60d)
- 日线振幅

### 1.2 定义"趋势期"

候选定义（需回测验证哪个最优）：

```python
# 方案A：MA + 方向一致性
trend_up = close > MA20 连续 N 天 AND close[today] > close[today-N]
trend_dn = close < MA20 连续 N 天 AND close[today] < close[today-N]

# 方案B：ADX + 方向
trend = ADX > 25 AND (DI+ > DI- 连续 N 天)

# 方案C：日线振幅收窄 + 方向
trend = 5日平均日线振幅 < 1.2% AND |5日累计涨跌| > 3%

# 方案D：组合
trend = (close > MA20 连续 3天) AND (ADX > 20) AND (5日振幅 < 1.5%)
```

### 1.3 历史趋势期统计

对2015~2026的趋势期统计：
- 持续天数分布
- 总涨跌幅分布
- 最大回撤
- 趋势结束后的走势（反转概率）

### 1.4 与日内策略亏损期重合度验证

对215天（有日内回测数据的）：
- 趋势期 vs 日内策略亏损日的重合度
- 如果重合度 > 70%，说明识别有效

## Phase 2：入场/出场规则设计（1-2天）

### 2.1 入场信号

```python
# 多头入场
entry_long = (
    close > MA20                      # 价格在均线上方
    AND close > close[-3]             # 3天累计上涨
    AND ADX > 20 AND ADX_rising       # 趋势正在加强
    AND avg_daily_range(5) < 1.5%     # 日内振幅不大（排除震荡日）
)

# 空头入场（对称）
entry_short = (
    close < MA20
    AND close < close[-3]
    AND ADX > 20 AND ADX_rising
    AND avg_daily_range(5) < 1.5%
)
```

### 2.2 仓位管理

- 1手IM或IC（验证期间）
- 用锁仓方式持仓（避免平今手续费万分之2.3）
- 次日开盘平昨仓+开新仓（roll），手续费万分之0.23
- 持仓成本 ≈ 0.23/万 × 2 × 每天 ≈ 可忽略

### 2.3 出场规则

| 优先级 | 出场条件 | 说明 |
|--------|---------|------|
| 1 | 日线收盘跌破MA20 | 趋势结束 |
| 2 | ADX从>25回落至<18 | 趋势减弱 |
| 3 | 单日亏损>1.5% | 固定止损 |
| 4 | 持仓满10天 | 时间止损（防死扛） |
| 5 | 日内振幅突然>2.5% | 市场转为震荡，切回日内策略 |

### 2.4 与日内策略协调

```
场景1：波段持多 + 日内信号做空
  → 日内做空信号正常执行（锁仓），相当于部分对冲
  → 日内平仓后恢复净多头

场景2：波段持多 + 日内信号做多
  → 日内加仓（但总仓位不超过限制）

场景3：波段持仓期间日内振幅放大
  → 如果日内振幅 > 2%，波段策略出场，切回日内模式
```

## Phase 5：手续费优化（0.5天）

实际成本估算：
- 开仓：0.23/万 × 200 × 7500 = ~3.5元/手
- 每日roll：3.5 × 2 = ~7元/天
- 10天持仓：~70元手续费
- 对比预期收益：1400pt × 200 = 28万元 → 手续费忽略不计

## 风险

1. **A股期货趋势持续性**：可能不如海外CTA标的，频繁假突破
2. **入场滞后**：趋势确认时可能已走了30-50%
3. **锁仓占保证金**：波段+日内同时需要更多保证金
4. **协调复杂度**：两个策略同时运行的状态管理
5. **回测偏差**：日线回测无法反映盘中的止损精度

## 成功标准

- Phase 1 识别出的趋势期与日内策略亏损日重合度 > 60%
- Phase 3 波段策略在11年中至少8年盈利
- Phase 3 联合回测（日内+波段）比单独日内在215天上多 > 500pt
