# 策略手册 — MO/IM 波动率周期交易

> 基于2026年3月实盘经验总结。核心思路：用IV水平决定买/卖方向，用市场方向决定Call/Put偏重。
>
> 最后更新：2026-04-02

---

## 一、市场环境象限体系

用三个维度划分市场状态（2026-04-02升级，加入Hurst趋势/震荡维度）：

- 维度1：IV水平（ATM IV分位，P70+高/P30-低）
- 维度2：VRP方向（VRP>0卖方有利/VRP<0卖方不利）
- 维度3：Hurst指数（H>0.55趋势/H<0.45震荡）

```
高IV象限按Hurst细分：

  VRP>0（卖方有利）              VRP<0（卖方不利）
  ┌──────────────────┬──────────────────┐
  │ A1: 震荡(H<0.55) │ B1: 震荡(H<0.55) │
  │ → 卖方最佳甜点区 │ → 等VRP转正再开  │
  │ 卖Strangle赚Theta│ → 可小仓卖方     │
  ├──────────────────┼──────────────────┤
  │ A2: 趋势(H>0.55) │ B2: 趋势(H>0.55) │
  │ → 顺势卖方可以   │ → 最危险！禁止   │
  │ → 逆势端加保护   │ → 纯买方或对冲   │
  └──────────────────┴──────────────────┘

低IV象限不细分（Hurst影响较小）：
  C: 低IV+偏多 → 买Call / IM+Put吃贴水
  D: 低IV+偏空 → 卖Call Spread / 减少暴露
```

当前位置判断方法：
- IV水平：vol_monitor --snapshot → IV分位
- VRP/期限结构：vol_monitor → VRP值 + 期限结构形态
- Hurst：quadrant_monitor → Hurst(60d) + 历史分位
- 市场方向：daily_record.py eod → 连涨跌天数 + 20日范围位置

### 关于OTM距离的基准

本手册中所有"OTM X%"均指相对期货价格（或隐含Forward）的距离：
- 贴水市场中期货 < 现货，用现货算Put OTM会高估安全距离
- 例：期货7633，卖P6800 → OTM = 10.9%。同样用现货算是12.2%，多了1.3%的虚假安全感
- vol_monitor同时显示三个基准（Forward/期货/现货），Put端以期货为准
- Call端反过来：现货更高，用期货算Call OTM会高估安全距离，Call端以现货为准

## 二、各象限详细策略

### 象限A：高IV + 方向偏多（卖波动率最佳窗口）

识别：IV分位>P70，现货在20日范围中位以上，连涨2天+

主策略：卖OTM Strangle
- 卖P(OTM 8-10%) + 卖C(OTM 10-12%)
- Call端比Put端远2-3%，反映偏多方向判断
- 高IV两端权利金都肥，上涨+IV回落 = Delta和Vega双赚，Theta每天收钱

VRP分层：
- VRP > 0（IV > RV，期权真的贵）→ 裸卖Strangle，利润空间最大
- VRP < 0（IV高但RV更高，卖的是便宜保险）→ Put端改用Spread保护（卖P + 买更远P），封住最大亏损

仓位规则：
- 保证金 ≤ 15%，行权价距现货 ≥ 8%（红线），净Delta ±100以内

退出：IV分位降到P30 → 止盈 | 标的跌>3% → 止损Put端 | 到期前7天 → 换月

3/24-25实例：暴跌后IV=39%(P95+)，反弹开始。卖C8600+P6800(OTM 10%+)，两天浮盈15K→21K。但当时VRP=-12%（RV远高于IV），回头看Put端应该用Spread更稳妥。

CRO警告——暴涨踩踏风险：
- A股高IV后的反弹往往极其暴力（千股涨停级V型反转）
- IV在回落Vega赚钱，但标的单日暴涨3-5%时Call端遭遇Delta+Gamma双杀
- 战术修正：偏多环境下不对称构建，Call端OTM必须远大于Put端
- 激进做法：V型反弹初期只做单腿Put Credit Spread，彻底放弃Call端利润
- 宁可少赚Theta，不把脖子伸给暴涨的镰刀

---

### 象限B：高IV + 方向偏空（最危险也最有机会）

识别：IV分位>P70(通常>P85)，IV日变动>+3pp，现货在20日范围下沿，期限结构倒挂

注意区分两个子状态：
- B1: RV > IV（VRP<0，波动还在加剧）→ 买方赚Gamma
- B2: RV < IV（VRP>0，恐慌过度定价）→ 卖方赚Vega

子状态B1（恐慌进行中，RV > IV）：
- 主策略：买OTM Put做方向，或Long Straddle赚Gamma
- 赚钱靠Gamma（标的大幅波动的非线性利润），不是靠Vega
- 即使之后IV回落Vega亏钱，只要波动够大Gamma能覆盖
- 辅助：Gamma Scalping（IM多头+买Put，贴水补贴Theta）
- 绝对禁止：卖Put、抄底做多期货、加仓卖方仓位

子状态B2（恐慌见顶，IV开始回落但仍在高位）：
- 主策略：卖OTM Strangle，赚IV回落的Vega
- 此时IV从peak往下掉是确定性最高的钱
- Call端选远（方向可能反弹），Put端OTM 10%+
- 这就是3/24-25的实际操作：IV从39%回落中卖C8600+P6800

退出B1：IV从peak回落>5pp 或 标的单日反弹>3% → 切换到B2
退出B2：IV分位降到P50以下 → 切换到象限A正常运营

B1→B2切换铁律（全系统最难的动作）：
- 不能只看IV回落，必须同时看RV收敛
- 标的单日振幅仍>3%时，哪怕IV分位已降到P80，也不切换到卖方
- VRP < 0 是不可逾越的红线：RV > IV时绝不卖期权，权利金再贵也不行
- 安全切换条件：IV从peak回落>5pp + 连续2天日振幅<2% + VRP转正

3/23实例（B1）：暴跌-4.8%，IV=39%但RV=50%+，VRP=-3.29%。被迫清卖方-77,580。保留P7600多头+29,700。
3/24-25实例（B2）：IV从39%开始回落，反弹中卖C8600+P6800，两天浮盈15K→21K。

3/20的教训：VRP=BUY_VOL不是说"去买straddle"，而是说"别再卖了，卖方在亏钱"。当天浮亏-85K仍持有卖方仓位，忽略了系统警告。

### 象限C：低IV + 方向偏多（低成本做多窗口）

识别：IV分位<P30(约22%)，VRP<0(IV低于RV)，贴水年化>8%，现货从底部反弹或缓慢爬升

主策略1：IM期货+买Put（贴水捕获）
- IM贴水年化10%+提供外生Theta
- 买OTM 5%的Put做保护，Gamma Scalping对冲Delta
- 系统回测：Gamma/Theta覆盖率106%，Put保护基本免费
- 适合持有2-4周，到期前5-7天换月

主策略2：买远月OTM Call
- IV低时Call权利金便宜，买入MO2606或更远月份
- 选OTM 3-5%，Delta约0.3-0.4
- 用少量资金（账户0.5%）博方向+波动率双升

辅助：卖近月高IV的Put Spread
- 如果近月IV仍比远月高（倒挂），卖近月Put Spread+买远月Call = 用Theta养Call

不适合：纯卖Strangle（IV太低，Theta收入不值得冒风险）

退出：IV分位回到P50+ → 考虑止盈买方转卖方 | 贴水收窄<5% → IM+Put策略减仓

理想切换触发：从象限A的卖方 → 平仓止盈 → 进入象限C的买方。两个条件同时满足：IV<P30 + 贴水年化>8%。

合约乘数配比（关键实操细节）：
- IM期货乘数=200，MO期权乘数=100，比例2:1
- 1手IM期货需要2手MO期权才能Delta匹配
- Gamma Scalping的对冲计算必须严格按乘数比例换算
- 比例不对会导致对冲网出现系统性敞口漏洞
- 在这个象限，耐心是最大的武器——只要贴水还在，外生Theta就持续补贴保护成本

---

### 象限D：低IV + 方向偏空（最难的行情）

识别：IV分位<P30，RV也低（日均波动<1%），现货缓慢阴跌（每月跌2-3%），无恐慌

主策略：卖Call Spread（顺势收Theta）
- 卖C(OTM 8-10%) + 买C(OTM 15%)保护
- 阴跌方向让Call端安全远离，Theta稳定收入
- 不卖Put——方向向下，Put端风险在增加

辅助：少量远月便宜Put（防止阴跌变暴跌）
- 用Theta收入的10-20%买远月OTM 15%的Put
- 保险性质，大部分时候会亏光，但防黑天鹅

不适合：
- 卖Strangle（Put端会越来越不安全）
- 买Straddle（波动不够大赚不到Gamma）
- IM+Put（慢速阴跌IM多头天天亏，Gamma不够补）

退出：标的跌>5%且IV开始飙升 → 象限切换到B，转买方 | 持续横盘>2周IV降到历史低位 → 少量买Straddle

现实：这个象限赚钱难度最高，最好的策略可能是减少交易、缩小仓位、等待环境改善。

CRO特别警告——雪球敲入跃迁风险：
- A股"低IV阴跌"往往是流动性枯竭的前兆
- 雪球产品集中敲入时，会从象限D直接闪现跃迁到B1（暴跌+IV井喷）
- 没有经过A或C的过渡，传统的象限切换逻辑来不及反应
- 这就是远月虚值Put存在的唯一意义——99%时间亏钱，但D→B1跃迁时能救命
- Call Spread的Theta收入拿出20%买彩票Put，是沼泽地唯一的生存方式

## 三、象限切换与过渡操作

市场不会跳变，象限之间有过渡期。关键切换路径：

### A→B（上涨突然转暴跌）
- 触发：单日跌>3%，IV急升>5pp
- 操作：立即止损卖Put端，保留卖Call端（方向顺风），考虑买Put对冲
- 3/23就是这个切换——但当时卖方仓位太重，止损代价太大

### B→A（恐慌消退，反弹开始）
- 触发：IV从peak回落3pp+，现货连涨2天
- 操作：平掉买方止盈，逐步建立卖方仓位
- 先1手Strangle试水，2-3天内确认方向后加到目标仓位
- 检查VRP：转正则裸卖，仍为负则Put端用Spread

### A→C（IV持续回落到低位）
- 触发：IV分位从P70+降到P30-，Strangle利润接近最大值
- 操作：止盈平仓卖方Strangle，转IM+Put或买远月Call
- 不要等到IV完全到底——P40开始减卖方，P30建买方
- Strangle在IV低位时Theta收入太薄，不值得继续持有

### D→B（阴跌变暴跌，黑天鹅）
- 触发：单日跌>4%，IV飙升
- 操作：远月保护Put此时爆赚，卖Call Spread可以保留（安全），加买Put
- 这是保护Put存在的唯一意义——99%的时间在亏钱，但这1%能救命

## 四、仓位管理铁律

从3/20-3/23的教训中提炼，不可违反：

### 红线规则（任何时候）
1. 卖方行权价距现货 ≥ 8%，不例外
2. 保证金占用 ≤ 20%（极端情况临时到25%，当天必须减回20%）
3. 单策略最大亏损 ≤ 账户2%/天（约13万），触发强制减仓一半
4. 不同时持有期货多头和卖Put（方向暴露叠加，3/20教训）
5. VRP信号=BUY_VOL时，不新开卖方仓位

### 建仓规则
- 新策略先1手试水，盈利后2-3天内逐步加到目标仓位
- 单次加仓不超过现有仓位的50%
- 跨月份持仓要考虑换月风险，到期前5-7天开始roll

### Greeks控制
- 净Delta：±200以内（约±1手IM的暴露）
- 净Vega：卖方状态下-10,000以内，买方状态下+10,000以内
- 净Theta：正常范围+1,000~+3,000/天（卖方），不追求过高Theta

### 日内交易手续费
- 股指期货（IM/IF/IH/IC）平今手续费万分之2.3（开仓/平昨的10倍）
- 日内平仓必须用锁仓（反向开仓），次日再平昨仓
- order_executor 自动处理锁仓逻辑，次日启动时提醒平仓
- 期权（MO/IO）无平今加倍，正常平仓即可

### 换月与流动性枯竭（MO微观结构）
MO深度虚值期权（OTM>10%）在到期前10天左右会出现流动性枯竭：
- Bid-Ask Spread从0.2突然扩大到2.0甚至5.0
- 挂单可能完全没有对手方

卖方换月规则：
- OTM>10%的腿：到期前10-12天开始关注流动性，到期前7天必须完成roll
- 如果已赚到最大理论利润的80%+，距到期10天时直接吃盘口平仓
- 不要贪最后一点Theta——流动性干涸的滑点会吃掉一周的Theta收入

买方（彩票Put）规则：
- 到期前7天仍是深度虚值 → 直接放弃，让它归零
- 不要尝试平仓，手续费+滑点的成本可能比残值还高

实操检查：
- 每天vol_monitor检查持仓合约的盘口深度（bid_volume/ask_volume）
- 单边挂单量<10手时视为流动性预警，启动换月

## 五、日内信号系统详解

### 信号版本

- v2：适用于IM/IF/IC，动量突破型，追踪趋势启动
- v3：适用于IH，均值回归型（IH自相关接近0，动量策略不适用）

### 信号得分构成

总分 = (M + V + Q) × d × t × s

满分100分，阈值60分。各维度：

#### M — 动量得分（0~50分）

衡量价格是否出现方向性突破：
- 5分钟MACD方向和强度
- 价格相对VWAP的偏离程度（高于VWAP偏多，低于偏空）
- 布林带位置（突破上轨偏多，突破下轨偏空）
- 多周期EMA排列（5/10/20 EMA多头或空头排列）

M=50：多个指标共振确认同一方向，强突破
M=30：部分指标确认，方向初现
M=0：价格在中间区域震荡，无明确方向

这是最重要的维度，没有M分基本不会触发信号。

#### V — 波动率得分（0~30分）

衡量波动率是否在扩张（趋势启动的前兆）：
- ATR短周期(5根) / ATR长周期(20根) 的比值
- 比值>1说明近期波动加速，趋势可能正在形成

V=30：短期波动率显著高于长期，波动在扩张
V=15：轻微扩张
V=0：波动平稳或收缩，无趋势迹象

V=0不代表信号一定差（统计验证WR=56%），因为趋势启动初期ATR还没来得及扩张。

#### Q — 成交量得分（0~20分）

衡量当前K线是否放量（资金推动确认）：
- 当前5分钟K线成交量 / 近20根K线平均成交量
- 放量>1.5倍得高分，缩量<0.7倍得低分

Q=20：显著放量，资金在推动方向
Q=10：温和放量
Q=0：缩量，方向缺乏资金确认

早盘前30分钟成交量天然偏高，Q分需谨慎解读。

#### d — 日线方向乘数（0.5~1.2）

衡量日内信号方向是否和日线趋势一致：
- 用最近5天的日线动量（现货指数，非期货）判断日线方向
- 信号方向与日线一致 → d=1.2（顺势加成20%）
- 信号方向与日线中性 → d=1.0（不调整）
- 信号方向与日线冲突 → d=0.5（逆势重罚50%）

d=0.5的统计依据：30天回测中逆势做空信号0%胜率、平均-11.8pt，原来d=0.7仍有漏网，加强到0.5后这类信号全部被过滤。

盘前Morning Briefing可以覆盖d值：当briefing判断今天方向和5日日线动量不同时（比如日线偏空但外盘暴涨判断今天反弹），通过d_override调整。

#### t — 趋势完整度乘数（0.5~1.2）

衡量多时间框架是否确认同一方向：
- 5分钟方向 + 15分钟方向同时确认 → t=1.2（双确认）
- 只有5分钟确认 → t=1.0
- 5分钟确认但15分钟不确定 → t=0.7
- 5分钟和15分钟方向矛盾 → t=0.5

面板中的5m/15m列：
- ^L = 做多确认，vS = 做空确认
- -- = 方向不确定

#### s — 情绪乘数（约0.7~1.15）

基于期权市场数据调整信号强度：
- ATM IV日变动：IV急升>3pp时s降低（恐慌环境信号不可靠）
- VRP方向：VRP<0时s降低（市场定价混乱）
- 25D Risk Reversal：Skew极端时s降低
- 期限结构：倒挂时s降低（-0.05）
- PCR：极端值时调整

s的影响通常在0.85~1.05之间，微调作用，不会大幅改变信号。

### 面板读法示例
```
IM  | 7757.0 | 7689.2 | +0.74% | -0.64 | 56 | ^L | ^L | MID+ | ^ 21 |
     v2[M0/V30/Q0/d1.0/t0.7]
```

从左到右：
- 现货7757 / 期货7689 / 偏离+0.74% / Z-Score -0.64 / RSI 56
- 5分钟做多(^L) / 15分钟做多(^L) / 布林中轨上方(MID+)
- 方向做多(^) / 过滤后得分21
- 原始分M0+V30+Q0=30，× d1.0 × t0.7 = 21，低于60不触发

### 信号触发条件

同时满足以下全部条件才触发SIGNAL：
1. 过滤后得分 ≥ 60
2. 在开仓时间窗口内（09:45~14:15）
3. 不在午休禁开期（11:20~13:05）
4. 不在冷却期（同一品种上次信号后至少2根K线）
5. 同一根K线不重复触发

### 关键参数总结

| 参数 | 当前值 | 数据依据 |
|------|--------|---------|
| 信号阈值 | 60 | 敏感性分析：thr60在所有滑点水平下最优 |
| 逆势惩罚d | 0.5 | 逆势做空0%胜率，平均-11.8pt |
| 开仓窗口 | 09:45~14:15 | 09:45避开盘噪音，14:15留至少45分钟 |
| 确认超时 | 30秒 | 超时自动记为NOTED |
| Position sizing | Fixed Risk 0.5% | 5种方法对比最优，PnL×3且MaxDD<1% |
| 建议手数 | sizing ÷ 2 | 保守起步，积累数据后再调整 |

## 六、系统信号与人工判断的配合

### 系统能做好的（信任系统）
- IV分位和VRP信号：判断买/卖波动率的方向
- 持仓安全距离监控：行权价预警
- Greeks实时计算：暴露量化
